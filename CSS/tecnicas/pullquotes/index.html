Pullquotes em CSS

Olhos (que os gringos chamam de "pullquotes") é um recurso para diagramação de revistas e jornais antigo e muito utilizado até hoje. Trata-se daqueles textos que são colocados no decorrer de um artigo ou uma matéria geralmente destacando alguns trechos e tornando a leitura menos densa. Algumas pessoas já o trouxeram para a web mas sua utilização parece ainda não ser unânime. Fica dada a dica.

Muito interessante podemos desenvolver somente com as CSS. Técnicas e técnicas surgem a cada dia.

Dessa forma eu não preciso utilizar HTML extra e desnecessário para utilizar este recurso visual. No HTML eu apenas coloquei uma classe (com o valor de “pullquote”) no elemento que me interessa destacar como um olho. Semanticamente eu considero meu pullquote (ou “olho”) apenas como uma ênfase (strong) e não como a citação de um trecho (blockquote). Por isso eu utilizo o <strong> com uma classe da seguinte maneira.

Você encontra vários artigos e tutoriais sobre este assunto que é bem simples de implementar no seu site. Alguns utilizam JavaScript outros apenas CSS para reproduzir estes pullquotes; você escolhe. Eu acabei de implementar aqui apenas uma única regra de CSS que faz tudo o que eu preciso.
