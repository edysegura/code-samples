Scripts Greasemonkey

A extensão greasemonkey permite modificar, alterar, melhorar, corrigir e tudo o que for possível ser feito através de javascript em uma página web.
Para instalar a extensão use a página de add-ons do firefox.

Oi Tab

Script para ordenar a tabulação no Oi Torpedo. Instalar script oitab.user.js

No Overlib

Script para tirar a popup chata do dotproject. Instalar script no.overlib.user.js

Auto Complete DotProject

Script para login do dotproject. Instalar script no.overlib.user.js

Select User

Script para logs do dotproject. Instalar script selectuser.user.js
